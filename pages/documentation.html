Welcome to the Multiply.js website.

Welcome to the Multiply.js website. Multiply.js is a javascript library for math related things, from processing data sets, or just basic calculations. This javascript library is easy to use, and can be used with any other library.

Usage

Multiply.js is extremely easy to use and very compatible. Not only is Multiply.js free to use, but it is also open source. You can go to Github, and fork the repository.
